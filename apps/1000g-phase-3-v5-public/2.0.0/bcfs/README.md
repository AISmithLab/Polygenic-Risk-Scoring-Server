# bcfs/

Per-chromosome BCF files from the 1000 Genomes Phase 3 v5 reference panel, used by Eagle2 for haplotype phasing.

## Expected files

One BCF + CSI index per chromosome (1–22, X):

```
ALL.chr1.phase3_v5.shapeit2_mvncall_integrated.noSingleton.genotypes.recode.bcf
ALL.chr1.phase3_v5.shapeit2_mvncall_integrated.noSingleton.genotypes.recode.bcf.csi
...
ALL.chrX.phase3_v5.shapeit2_mvncall_integrated.noSingleton.genotypes.recode.bcf
ALL.chrX.phase3_v5.shapeit2_mvncall_integrated.noSingleton.genotypes.recode.bcf.csi
```

## Source

Michigan Imputation Server reference panel distribution:
https://imputationserver.sph.umich.edu/start.html#!pages/refpanels
