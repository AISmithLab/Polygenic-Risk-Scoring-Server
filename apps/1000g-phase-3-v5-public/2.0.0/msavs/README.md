# msavs/

Per-chromosome MSAV files (Minimac4 reference panel format) used for genotype imputation. Each file encodes haplotype dosage and parameter estimates for one chromosome.

## Expected files

One MSAV per chromosome (1–22, X including PAR regions):

```
1.1000g.Phase3.v5.With.Parameter.Estimates.msav
2.1000g.Phase3.v5.With.Parameter.Estimates.msav
...
22.1000g.Phase3.v5.With.Parameter.Estimates.msav
X.1000g.Phase3.v5.With.Parameter.Estimates.msav
X.PAR1.1000g.Phase3.v5.With.Parameter.Estimates.msav
X.PAR2.1000g.Phase3.v5.With.Parameter.Estimates.msav
```

Michigan Imputation Server reference panel distribution:
