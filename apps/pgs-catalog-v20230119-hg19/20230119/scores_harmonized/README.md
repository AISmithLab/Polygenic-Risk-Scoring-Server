# scores_harmonized/

Intermediate harmonized score files produced during the PGS Catalog collection build for v20230119 (hg19). These are the per-score files as output by the harmonization pipeline before being merged into the `scores.txt.gz` collection.

## Expected file pattern

```
PGS<id>.txt.gz      # harmonized variant weights
```

## Note

The final merged collection used by the pipeline is `../scores.txt.gz` (generated by `pgs-calc create-collection`). This directory is an intermediate artifact and does not need to be repopulated if `scores.txt.gz` already exists.
